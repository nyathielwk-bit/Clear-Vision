A small pill for status and metadata — stock state, "New", membership tier, etc.

```jsx
<Badge tone="success" dot>In stock</Badge>
<Badge tone="accent">New</Badge>
<Badge tone="brass" variant="solid">Member</Badge>
<Badge variant="outline">Limited</Badge>
```

`tone`: neutral / accent / brass / success / warning / danger / info. `variant`: soft (default) / solid / outline. `dot` adds a leading status dot.
